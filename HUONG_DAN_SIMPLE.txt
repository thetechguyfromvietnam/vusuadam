CACH UPLOAD DATABASE LEN VERCEL
=====================================

1. TAO POSTGRES TREN VERCEL:
   - Vao Vercel Dashboard
   - Chon project -> Storage
   - Create Database -> Postgres
   - Chon Hobby plan (mien phi)
   - Copy Connection String

2. THEM ENVIRONMENT VARIABLE:
   - Settings -> Environment Variables
   - Name: POSTGRES_URL
   - Value: Connection string vua copy

3. MIGRATE DU LIEU:
   pip install psycopg2-binary
   export POSTGRES_URL='connection-string'
   python3 setup_vercel_postgres.py

4. DEPLOY:
   git push

XONG! Database se duoc upload len Vercel.
