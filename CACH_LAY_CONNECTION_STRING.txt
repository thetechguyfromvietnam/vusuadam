CACH LAY CONNECTION STRING TU SUPABASE
======================================

BUOC 1: Dang nhap Supabase
---------------------------
1. Vao: https://app.supabase.com
2. Dang nhap tai khoan cua ban

BUOC 2: Chon Project
---------------------
1. Trong danh sach projects, tim va click vao project: "supabase-gray-dog"
2. Hoac neu chua co project, click "New Project" de tao moi

BUOC 3: Vao Settings -> Database
---------------------------------
1. O sidebar ben trai, click vao "Settings" (icon bÃ¡nh rÄƒng)
2. Trong menu Settings, click vao "Database"

BUOC 4: Tim Connection String
------------------------------
1. Scroll xuong phan "Connection string"
2. Ban se thay nhieu loai connection string:
   
   a) URI (Session mode):
      - Format: postgresql://postgres:[YOUR-PASSWORD]@db.xxx.supabase.co:5432/postgres
      - Day la connection string chinh
   
   b) Connection pooling:
      - Format: postgresql://postgres.xxx:[YOUR-PASSWORD]@aws-0-xxx.pooler.supabase.com:6543/postgres
      - Khuyen nghi cho production

BUOC 5: Copy Connection String
-------------------------------
1. Click vao nut "Copy" ben canh connection string
2. Hoac highlight va copy (Cmd+C / Ctrl+C)
3. LUU Y: Thay [YOUR-PASSWORD] bang password that cua ban
   - Neu chua biet password, click "Reset database password"
   - Hoac xem trong "Database" -> "Connection info"

BUOC 6: Kiem tra Connection String
-----------------------------------
Connection string phai co dang:
postgresql://postgres:MAT_KHAU_THAT@db.xxx.supabase.co:5432/postgres

Trong do:
- postgres: username
- MAT_KHAU_THAT: password that cua ban (khong phai [YOUR-PASSWORD])
- db.xxx.supabase.co: host cua Supabase
- 5432: port
- postgres: database name

VI DU:
postgresql://postgres:Abc123xyz@db.abcdefgh.supabase.co:5432/postgres

LUU Y:
- KHONG duoc de [YOUR-PASSWORD] trong connection string
- Phai thay bang password that
- Neu quen password, reset lai trong Settings -> Database
